2. 内部処理

2.1 インストール時の内部処理

AppCmd.exeによる自動登録方法:
%windir%\system32\inetsrv\appcmd set config /section:isapiCgiRestriction /+"[path='C:\inetpub\wwwroot\bbb.exe',description='TakeUrl CGI',allowed='True']"

%windir%\system32\inetsrv\appcmd set config /section:handlers /+"[name='TakeUrlHandler',path='/aaa',verb='GET,HEAD,POST',modules='CgiModule',scriptProcessor='C:\inetpub\wwwroot\bbb.exe']"

AppCmd.exeによる自動解除方法:
%windir%\system32\inetsrv\appcmd set config /section:isapiCgiRestriction /-"[path='C:\inetpub\wwwroot\bbb.exe',description='TakeUrl CGI',allowed='True']"

%windir%\system32\inetsrv\appcmd set config /section:handlers /-"[name='TakeUrlHandler']"
